2 연립방정식

  1. 질문을 한글로 입력
  2. 질문을 영어로 번역
  3. 영어번역된 질문을 chatGPT에 문의
  4. 영문으로 작성된 chatGPT결과를 한글로 번역
  5. 최종 한글 번역결과를 출력결과로 표시

2.1 한글질문 준비

library(tidyverse)
library(openai)

korean_question <- "중학교에서 수학에서 나오는 연립방정식을 설명해줘"
korean_question
#> [1] "중학교에서 수학에서 나오는 연립방정식을 설명해줘"

2.2 질문번역


korean_question_input <- glue::glue("translate it into English: {korean_question}")

question_model <- create_completion(
    model="text-davinci-003",
    prompt = korean_question_input,
    max_tokens=1024,
    openai_api_key = Sys.getenv("OPEN_AI_KEY")
)

translated_questions <- str_extract(question_model$choices$text, "\\b[^\\W].+\\b")

translated_questions
#> [1] "Explain the system of linear equations that come from middle school math"

2.3 chatGPT 영문 답변


answer_model <- create_completion(
    model="text-davinci-003",
    prompt = translated_questions,
    max_tokens=1024,
    openai_api_key = Sys.getenv("OPEN_AI_KEY")
)

chatGPT_answer <- answer_model$choices$text
cat(chatGPT_answer)
#> 
#> 
#> Linear equations are used to represent linear relationships between two variables. In mathematics, a linear equation is an equation that describes a straight line. Examples of linear equations include y=x+3, x+2y=3, 3x2+2=6x+5, etc. In middle school math, linear equations are used to solve various problems involving geometry, linear functions, and basic algebraic equations. For example, a student may be asked to solve the equation y=2x+3 to find the value of x when y=4. Linear equations can be manipulated using the four basic operations (addition, subtraction, multiplication, and division) in order to isolate the variable of interest (in this case, x).

2.4 chatGPT 답변 번역


chatGPT_answer_request <- glue::glue("한국어로 번역해주세요: {chatGPT_answer}")

return_model <- create_completion(
    model="text-davinci-003",
    prompt = chatGPT_answer_request,
    max_tokens=1024,
    openai_api_key = Sys.getenv("OPEN_AI_KEY")
)

cat(return_model$choices$text)
#>  
#> 
#> 선형 방정식은 두 변수 간의 선형 관계를 나타내기 위해 사용됩니다. 수학에서, 선형 방정식은 직선을 설명하는 방정식입니다. 선형 방정식의 예로는 y=x+3, x+2y=3, 3x2+2=6x+5 등이 있습니다. 중학교 수학에서는 선형 방정식이 지리학, 선형 함수 및 기본 대수 방정식에 관련된 다양한 문제를 해결하는 데 사용됩니다. 예를 들어, 학생에게 y=2x+3 과의 방정식을 해결하여 y=4가 되는 x 값을 찾아라고 묻을 수 있습니다. 선형 방정식은 4가지 기본 연산(덧셈, 뺄셈, 곱셈, 나눗셈)을 이용하여 관심이 있는 변수(이 경우 x)를 분리 시킬 수 있습니다.
